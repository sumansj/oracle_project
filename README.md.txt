Aim:
Its a Oracle based project to analysis the sales data.

Technology Used:
Oracle 19c
Sql Developer 21.4

Sales Data Analysis of a company
--------------------------------------
1. Create tables(Customers, Stores, Products, Orders, Delivery) to store data
2. Implementating insert query for inserting records into the tables.
3. Implementing different types of Joins and Operators and understanding the difference between them.
4. Selecting records based on Pattern matching and wild card operators and working on them.
5. Analysing the data using Sub query, Group by clause along with aggregate functions.
6. Use different SQL Server functions such as REGEXP_LIKE(),REGEXP_REPLACE(),MAX(),MIN(),COUNT()
7. Implement update query to perform update task.
8. Use Triggers and Sequence.
